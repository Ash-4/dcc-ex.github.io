This page outlines how to perform a release, as well as provides guidelines on what is required before a release can be done.

Release Requirements
====================

These items must be complete prior to creating a release.

- Version is up to date including notes of what has changed
- Release notes up to date with the changes in this release
- A DCC-EX News article has been prepared
- The majority of the DCC-EX leadership team have agreed the release is ready

Creating a Release
==================

- Navigate to the GitHub repository
- If the release is to be created for an existing tagged version, go to that tagged version (how?)
- Download the ZIP of the repository
- 
- Navigate to the Releases tab
